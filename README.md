# Ship Broker

Консольная стратегия о морской брокерской компании.  
Управляй флотом, выбирай рейсы, рассчитывай прибыль и справляйся с неожиданными событиями.

## Особенности игры
- Управление портфелем судов
- Выбор рейсов с разными доходами и рисками
- Расчёт прибыли и убытков
- Случайные события: штормы, задержки, колебания цен
- Пошаговый игровой процесс с текстовым интерфейсом

## Требования
- Python 3.10+
- Библиотека `rich` для красивого текста

```bash
pip install rich

## Установка

### Клонируем репозиторий
git clone <ссылка на репозиторий>
cd ShipBroker

### Создаём виртуальное окружение
python3 -m venv venv
source venv/bin/activate

### Устанавливаем зависимости
pip install -r requirements.txt

## Запуск

python main.py
